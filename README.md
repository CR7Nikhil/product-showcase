# Product Showcase Website

A responsive product showcase website built using HTML, CSS, and JavaScript

## 🚀 Technologies Used
- HTML5
- CSS3 (Responsive Design & Media Queries)
- JavaScript (DOM Manipulation & Interactivity)

## 📌 Features
- Clean and structured layout
- Fully responsive design
- Interactive product display
- Optimized JavaScript performance

## 🧠 Challenges Faced
1. Ensuring consistent responsiveness across devices  
   - Solved using CSS media queries and layout restructuring.

2. Optimizing JavaScript performance  
   - Reduced unnecessary DOM manipulation and organized functions efficiently.

## 📂 Project Structure
- index.html – Main structure
- css/style.css – Styling and responsiveness
- js/script.js – Interactive behavior
